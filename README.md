# Due Again

Due Again is a local-first iOS app for tracking real-life upkeep tasks that become due again after a set number of days from the last completion.

Core idea: never wonder when you last did it.

## App Features

- SwiftUI iPhone app
- SwiftData local persistence
- Today, Coming Soon, All, and Settings tabs
- Local notification scheduling
- WidgetKit extension for due-again snapshots

## Project Structure

- `Due Again/` main app target
- `Due AgainWidget/` widget extension
- `Due AgainTests/` unit tests
- `Due AgainUITests/` UI smoke tests
